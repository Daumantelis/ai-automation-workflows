# ai-automation-workflows
Automation workflows using n8n, Make and AI integrations for process optimization and lead qualification.

## Real Workflow Examples

This repository includes real automation workflows:

- AI Translation QA automation with scoring and validation
- Batch-based translation processing pipeline
- Power Automate web automation workflow
- API-driven automation with Supabase and Lokalise

All workflows demonstrate real-world automation architecture including batching, retry logic, state tracking and external API integration.
